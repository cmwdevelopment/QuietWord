# QuietWord Keyboard Shortcuts

Quick reference for keyboard navigation in QuietWord.

## Reader Page (`/reader/john` or `/reader/psalm`)

| Key | Action | Notes |
|-----|--------|-------|
| `‚Üí` | Next chunk | Navigate to next reading chunk |
| `‚Üê` | Previous chunk | Navigate to previous reading chunk |

### Smart Behavior

The shortcuts are **intelligently disabled** when:
- You're at a checkpoint (typing in note field)
- An input or textarea is focused
- You've reached the first/last chunk (prevents navigation past boundaries)

## Future Shortcuts (Not Yet Implemented)

Ideas for power users:

- `J` / `K` - Vim-style navigation (down/up)
- `N` - Jump to Notes page
- `S` - Jump to Settings page
- `?` - Show keyboard shortcuts help
- `Ctrl+S` - Quick save note at checkpoint

---

## Tips

1. **Use arrow keys for fastest reading** - No need to click buttons
2. **ESC is your friend** - Quick exit from any page
3. **Shortcuts work on both desktop and laptop keyboards**
4. **Mobile users** - Touch navigation buttons work great too!
